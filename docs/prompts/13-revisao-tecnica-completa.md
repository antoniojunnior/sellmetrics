Você irá realizar uma revisão técnica completa do projeto Sellmetrics.
O objetivo é identificar fragilidades, inconsistências e oportunidades
de melhoria antes de qualquer uso em produção.

Percorra o projeto inteiramente e avalie cada camada abaixo:

CAMADA 1 — Modelo de dados e migrations
CAMADA 2 — Repositórios de acesso a dados
CAMADA 3 — Serviço de cálculo de indicadores
CAMADA 4 — Pipeline de ingestão
CAMADA 5 — Frontend e rotas
CAMADA 6 — Segurança geral
CAMADA 7 — Consistência documental

ENTREGÁVEL ESPERADO:
Produza um relatório estruturado com: CRÍTICO, IMPORTANTE, MELHORIA e OK.
Para cada item CRÍTICO e IMPORTANTE, inclua Descrição e Correção sugerida.
Após o relatório, corrija automaticamente todos os itens classificados como CRÍTICO.
